# Master Board

## Board name

`Grant Portal Program Board`

## Epics

Create and track epics using the exact names below:

1. `P0-Alignment`
2. `P2-UX-Content-Data`
3. `P3-App-Flow`
4. `P4-Auth-Account`
5. `P5-Support-Legal`
6. `P6-QA-Launch`

## Usage notes

- All delivery tickets for the 8-week launch window must belong to one of the epics above.
- Tickets that span epics should be split so each ticket has one primary epic owner.
- Cross-epic blockers should be tracked in the dependency document and reviewed in the weekly dependency ceremony.
