# Definition of Done (DoD) Template

Use this template in every delivery ticket before moving to Done.

## DoD Checklist

- [ ] Acceptance criteria met and validated with evidence.
- [ ] Legal signoff complete (if needed).
- [ ] Telemetry verified (events added/changed tested end-to-end).
- [ ] Accessibility check completed (manual + automated where applicable).
- [ ] Rollback note documented (revert path + risk).
- [ ] Documentation updated (runbooks, decision log, and dependency/risk artifacts as needed).

## Completion note format

```md
### DoD evidence

- AC verification:
- Legal signoff:
- Telemetry verification:
- Accessibility check:
- Rollback note:
- Docs updated:
```
